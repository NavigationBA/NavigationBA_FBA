    Topic 4.2 - 数据列操作¶

    1. 数据框中列操作基础¶

    (1) 数据列操作函数¶

    对于数据框中的列操作，我们这里介绍两个主要的函数：

    • mutate() 函数：

      • 用于对数据框中的列进行修改或者创建新的列
      • 里面的参数是新的列名 = 计算表达式形式，其中计算表达式可以使用数据框中已有的列来进行计算

    • select() 函数：

      • 里面的参数是列名，用于选择数据框中需要保留的列，其他没有被选择的列会被丢弃
      • 还可以使用 -列名 的形式来选择需要丢弃的列

    我们先导入上一节的包和数据：

    (2) 数据列操作练习¶

    我们接下来在以下任务中练习这两个函数。

    首先，我们创建一个子数据框 returns

    • 先筛选 invested_capital 大于 0 的公司，因为我们后续要计算投资回报率
    • 因为如果 invested_capital 小于等于 0 的话，投资回报率就没有意义了
    • 再新建一个新列，投资回报率：roic = ebit / invested_capital
    returns <- asx_200_2024 |>
  # 筛选出投资资本大于 0 的公司，因为只有这些公司才能计算 ROIC
  filter(invested_capital > 0) |>
  # 计算 ROIC，ROIC 的计算公式是 EBIT 除以投资资本
  mutate(roic = ebit / invested_capital)

    之后，我们再进行数据筛选排序：

    • 按照投资回报率 roic 这个变量进行降序排序
    • 并且取出前 10 行数据
    • 最后只展示 fyear、conml、roic、ebit、invested_capital 这几列
    returns_top10 <- returns |>
  # 按照投资回报率 roic 这一列进行降序排序
  arrange(desc(roic)) |>
  # 取出前 10 行数据
  slice_head(n = 10) |>
  # 只保留 fyear、conml、roic、ebit、invested_capital 这几列
  select(fyear, conml, roic, ebit, invested_capital)

    2. 数据框列操作的进阶操作¶

    (1) 创建多个新列¶

    在 mutate() 函数中，我们不仅可以创建一个新列，还可以一次性创建多个新列：

    • 创建多个新列，只需在 mutate() 函数中用逗号分隔多个新列名 = 计算表达式即可

    • 例如，我们想要在 returns 数据框中创建两个新列：

      • roic：投资回报率，计算公式是 ebit / invested_capital
      • roic_percent：将 roic 这个变量转换成百分比的形式，计算公式是 roic * 100
    returns_with_percent <- returns |>
  mutate(
    roic = ebit / invested_capital,
    roic_percent = roic * 100
  )

    (2) 丢弃列¶

    在 select() 函数中，我们可以选择需要保留的列，也可以选择需要丢弃的列：

    • 选择需要丢弃的列，在列名前加上 - 符号即可
    • 例如，我们想要丢弃 ebit 和 invested_capital 这两列，只保留其他列，我们就可以这样写：
    returns_selected <- returns |>
  select(-ebit, -invested_capital)

    3. 数据列操作小结¶

    在 R 的 dplyr 包中，数据列操作的主要函数是以下两个：

    • mutate() 函数：

      • 用于对数据框中的列进行修改或者创建新的列
      • 里面的参数是新的列名 = 计算表达式形式，其中计算表达式可以使用数据框中已有的列来进行计算
      • 可以一次性创建多个新列，只需在 mutate() 函数中用逗号分隔多个新列名 = 计算表达式即可

    • select() 函数：

      • 里面的参数是列名，用于选择数据框中需要保留的列，其他没有被选择的列会被丢弃
      • 还可以使用 -列名 的形式来选择需要丢弃的列
