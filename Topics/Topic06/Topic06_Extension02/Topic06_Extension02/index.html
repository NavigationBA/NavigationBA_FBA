    Topic 6 - Extension 2 - K-Means 与 PCA¶

    首先，我们先导入一下需要的包和数据：

    # 加载必要的包
library(tidyverse)
library(lubridate)
library(recipes)

# 读取数据与进行数据预处理
purchases <- read_csv("cdnow_purchases.csv")

customer_aggregates <-
  purchases |>
  group_by(users_id) |>
  summarise(
    last_purch_date = max(date),
    frequency    = n(),      
    monetary     = sum(amt),   
  ) |>
  ungroup() |> 
  mutate(
    # interval() 函数计算两个日期之间的时间间隔
    # %/% months(1) 将时间间隔转换为月数
    recency = interval(last_purch_date, analysis_date) %/% months(1)
    ) |> 
  select(users_id, recency, frequency, monetary)

rfm_data <-
  customer_aggregates |>
  mutate(
    r_tile = ntile(recency, 5),
    f_tile = ntile(frequency, 5),
    m_tile = ntile(monetary, 5)
  ) |> 
  mutate(
    # 将 R 的分数反转，使得 R = 5 代表最近购买的客户，R = 1 代表最久未购买的客户
    r_tile   = max(r_tile) - r_tile + 1,
  ) |>
  select(users_id, recency, r_tile, frequency, f_tile, monetary, m_tile)

# 选择 R、F、M 三个数值变量进行标准化处理
rfm_rec <- recipe(~ recency + frequency + monetary, data = rfm_data) |>
  step_normalize(all_numeric_predictors())

rfm_prepped <- prep(rfm_rec)

rfm_scaled <- bake(rfm_prepped, new_data = NULL)

# 设置K-Means模型
set.seed(123) 

kmeans_3 <- kmeans(rfm_scaled, centers = 3)

    如果我们使用 factoextra 包中的 fviz_nbclust() 函数来评估不同 K 值的聚类效果，我们会得到以下结果：

    library(factoextra)

fviz_cluster(
  kmeans_3, 
  data = rfm_scaled,
  geom = "point",
  palette = "jco",
  ggtheme = theme_minimal()
)

    这个结果是比较诡异的：

    • 因为我们数据其实是有三个维度的（Recency、Frequency、Monetary）
    • 但是我们在二维平面上展示了聚类结果
    • 这个 X 轴和 Y 轴事实上已经不是原始数据了，而是经过 PCA 降维后的结果

    PCA 的核心思想其实就是数据的投影：

    • 比方说，我们有一个二维数据，我们想把它投影到一个一维的空间上
    • 事实上，就是把每个点，都垂直地投射到这条线上，投射的点就是每个数据降维的结果
    • 而 PCA 是众多投影中，能把数据分布得最开、信息损失最小的那个投影

    同样的思想，我们可以把三维数据投影到二维空间上：

    • 在三维空间中，我们进行数据降维，其实就是将每个数据点投影到一个二维平面上
    • 数据中的每个点，投影到二维平面上的点，就是我们在二维图表中看到的点
    • 在众多可能的二维平面中，PCA 选择了那个能最大程度上保留数据分布特征的平面

    事实上，K-Means 和 PCA 没什么直接的关系：

    • K-Means 是一种聚类算法，主要用于将数据分成不同的类别
    • PCA 是一种降维算法，主要用于将高维数据投影到低维空间中，并且尽可能分得开
    • factoextra 包中的 fviz_nbclust() 函数的逻辑实际上就是，在评估 K-Means 聚类效果的时候，可以在原始的高维空间中进行评估，也可以在 PCA 降维后的空间中进行评估
